# 🪨 Rock Paper Scissors Game

A simple **Rock Paper Scissors** game developed in **Python**. This is a console-based mini project where the player competes against the computer, which makes a random move each time.

## ✨ Features

* Interactive command-line interface
* Play against the computer
* Random computer moves using Python's `random` module
* Automatic winner detection
* Handles invalid user input
* Displays game result with emojis

## 🛠️ Technologies Used

* Python 3
* `random` module

## 📂 Project Structure

```
rock-paper-scissors/

```

## ▶️ How to Run

1. Clone this repository.
2. Open the project folder.
3. Run the following command:

```bash
python rock_paper_scissors.py
```
